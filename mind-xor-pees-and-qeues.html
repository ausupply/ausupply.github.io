Logic Gates with Kick Drum and Oscilloscope

Oscilloscope Key

Steady Clock (Blue): A regular clock signal that toggles between 0 and 1 at a fixed tempo.

Random Clock (Green): A random signal that toggles between 0 and 1 based on a configurable probability.

Logic Output (Red): The result of applying the selected logic gate to the two input signals.

How It Works

This interactive tool demonstrates how logic gates work using two input signals:

  • Steady Clock (Blue): A regular clock signal that toggles between 0 and 1 at a fixed tempo.
  • Random Clock (Green): A random signal that toggles between 0 and 1 based on a configurable probability.

The selected logic gate combines these two signals to produce an output, which is displayed as a Logic Output (Red) signal. When the logic output is 1, a kick drum sound is played.

How to Use

  • Select a Logic Gate: Use the dropdown menu to choose between AND, OR, XOR, NAND, NOR, and NOT gates.
  • Adjust the Steady Clock Tempo: Use the "Steady Clock Tempo" slider to control the speed of the steady clock.
  • Adjust the Random Clock Probability: Use the "Random Clock Probability" slider to control how often the random clock toggles.

Logic Gates Explained

  • AND: Outputs 1 only if both inputs are 1.
  • OR: Outputs 1 if at least one input is 1.
  • XOR: Outputs 1 if the inputs are different.
  • NAND: Outputs 0 only if both inputs are 1 (opposite of AND).
  • NOR: Outputs 0 if at least one input is 1 (opposite of OR).
  • NOT: Outputs the opposite of the input (only uses the steady clock).
